To run, download:
- TextBlob 
  	pip install -U textblob
  	python -m textblob.download_corpora lite
  	- or -
  	git clone https://github.com/sloria/TextBlob.git
  	python setup.py install

- Tweepy, tabulate
  pip install tweepy
  pip install tabulate

Then, type "python deeplocal.py" into terminal.
